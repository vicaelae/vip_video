# vip_video

本仓库为脚本[VIP视频解析](https://scriptcat.org/script-show-page/792)的对接网站和脚本源码<br>
<u>本项目仅供学习使用</u>
本项目部署在基于IPFS开发的静态页面托管平台4everland<br>
理论上永远打得开<br>
可以使用GET方式传参<br>

<hr>
现版本支持：<br>
  - 脚本：爱奇艺、腾讯视频<br>
  - 网站：爱奇艺、腾讯视频<br>
